# Custom-card "Scenes Card"

The `card_scenes` you can control up to 5 `entity` such as `scene`, `script` or `automation`.

## Usage

```yaml
- type: "custom:button-card"
  template:
    - card_scenes
  variables:
    entity_1:
      entity_id: "scene.YOUR_SCENE"
      icon: "mdi:YOUR_ICON"
      icon_color: "yellow"
      name: "YOUR_NAME"
      bg_color: "yellow"
    entity_2:
      entity_id: "scene.YOUR_SCENE"
      icon: "mdi:YOUR_ICON"
      icon_color: "blue"
      name: "YOUR_NAME"
      bg_color: "blue"
    entity_3:
      entity_id: "scene.YOUR_SCENE"
      icon: "mdi:YOUR_ICON"
      icon_color: "purple"
      name: "YOUR_NAME"
      bg_color: "purple"
    entity_4:
      entity_id: "scene.YOUR_SCENE"
      icon: "mdi:YOUR_ICON"
      icon_color: "green"
      name: "YOUR_NAME"
      bg_color: "green"
    entity_5:
      entity_id: "scene.YOUR_SCENE"
      icon: "mdi:YOUR_ICON"
      icon_color: "red"
      name: "YOUR_NAME"
      bg_color: "red"
```
